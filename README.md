# Python校园二手交易平台

## 核心功能
- 用户管理
- 商品分类
- 发布商品
- 商品详情
- 最新发布
- 立即购买
- 联系卖家
- 支付订单
- 登录注册

## 适用场景
- 校园内学生之间二手物品交易
- 教师与学生之间教辅资料交换
- 校园周边商家与师生之间的物品交流

## 技术特点
- 基于Django框架，提供快速开发能力
- 使用MySQL数据库进行数据存储
- 前端采用HTML、CSS、JS等技术实现交互界面
- 支持远程调试，保证系统稳定性
